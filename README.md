# The Agency TLV

Premium real estate website built with React + Vite + Tailwind CSS.

## Stack
- React 18 + React Router
- Tailwind CSS + shadcn/ui
- Framer Motion
- Vite

## Setup

```bash
npm install
npm run dev
```

## Deploy to Vercel / Netlify
Push to GitHub and connect your repo. Build command: `npm run build`. Output: `dist`.
